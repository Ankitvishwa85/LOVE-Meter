# 💖 Love Compatibility Meter

Ek sundar aur interactive website jo **ladke aur ladki ke beech ke pyaar ko measure karti hai!**
Hindi aur English dono languages mein available hai! 🇮🇳 🇬🇧

---

## 🌟 Features

- 💖 **Love Score Calculator** — 0% se 100% tak compatibility score
- 🎂 **Date of Birth** — Automatically age calculate hoti hai
- 🎯 **Hobbies Match** — Clickable chips se select karo
- 💆 **Patience Level** — Slider se 1 se 10 tak
- 🍺 **Drinks & Smoking** — Yes/No toggle
- 💍 **Relationship Goal** — Marriage, Long Term, Casual, Friendship
- 👨‍👩‍👧 **We are Relative?** — Agar relative hain toh score cap ho jaata hai
- 📅 **Communication / Meetings** — Daily se Long Distance tak
- 🇮🇳 **Hindi / English Toggle** — Ek click mein poori website Hindi mein!
- 🎊 **Confetti Explosion** — Achhe score pe rang birangi confetti!
- 📊 **Animated Score Meter** — Sundar semicircle meter
- 📈 **Breakdown Bars** — Har category ka alag score

## 👨‍💻 Developer Notes

### CSS Variables
`:root` mein saare colors define hain — ek jagah badlo, sab jagah change ho jaayega!
```css
:root {
  --pink: #e8609a;
  --gold: #f5c842;
  --bg:   #0d0812;
}
```

### Language System
`LANG` object mein dono languages ka data hai:
```javascript
const LANG = {
  en: { /* English text */ },
  hi: { /* Hindi text  */ }
};
```

### Score Calculation
`calcScore()` function mein poora logic hai — har category ka score alag calculate hota hai!
